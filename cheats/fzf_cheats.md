### fzfでファイル検索の使い方
- 概要: fzfを使ってファイルや履歴をすばやく検索する  
- 操作:
  - `Ctrl + T` : ファイル検索して編集（使用例: vim, cd, cat など）
  - `Ctrl + R` : コマンド履歴の検索（pecoライク）

```zsh
# 補足:
# fzf単体では全ファイルを検索してしまうため、
# fdを組み合わせて検索範囲を限定するのが実用的。
- メモ:
 - `fd`コマンドをインストールし、`.gitignore`などを有効に不要なファイルを除外することを推奨
```
